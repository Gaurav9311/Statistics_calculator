# Statistics Calculator 📊

A command-line statistics calculator 
built with Python and NumPy.

## Features
- 📈 Mean
- 📊 Median  
- 📉 Variance
- 📐 Standard Deviation
- ⬆️ Maximum
- ⬇️ Minimum
- ➕ Sum
- 📏 Percentile

## How to Run
pip install numpy
python Statistics_calculator.py

## Tech Used
- Python 🐍
- NumPy
